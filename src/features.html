Features that fit how you work

From natural-language editing to GitHub PRs and cost transparency: the capabilities that make SiteBuilder useful for content teams and engineering alike.

Chat-based content editing

Describe changes in plain language; the AI applies them. Real-time streaming, live preview, markdown rendering, and prompt suggestions keep the flow smooth.

PhotoBuilder: AI images that match your page

No more prompting your way to a good photo. The AI analyzes your page and generates image prompts for you—so visuals fit tone, content, and layout from a single click. Launch from the content editor’s built-page list, tweak prompts if you like, then generate with OpenAI or Google Gemini, upload to S3, and embed without leaving the editor.

Optional dedicated API key; locale-aware prompts and regeneration with your own tweaks.

Cost transparency and BYOK

Bring your own LLM API key. See token usage and estimated cost as you chat. No surprise bills—you pay your provider directly.

Workspace isolation and security

Each workspace runs in an isolated Docker container with resource limits. Sandboxed filesystem, path traversal protection, and secure token storage keep your repos and keys safe.

Agent configuration

Per-project instructions, runtime choice (Node, Python, PHP, or custom Docker), and workspace rules in .sitebuilder/rules/ let you tailor the AI to your stack and conventions.

GitHub and pull request workflow

Connect your repo; each editing session gets its own branch. Send to review with one click and a PR is created. Reviewers can merge or reopen for changes.
